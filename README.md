#s3grep - grep multiple (gzipped) text files on s3

1. Written in Python3
2. Tested in Mac OS X El Captain, CentOS 7.0

##Release Notes

0.1 - First beta release, accept s3 url and regex as parameter, plenty of
features are pending

##Requirements
1. Python3.4+
2. boto3
